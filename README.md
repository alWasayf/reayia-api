# Reayai API

## Features
1. Login
2. Logout
3. Add babySitter
4. Update babySitter
5. Delete babySitter
6. View all babySitter 
7. Search babySitter by nationality
8. Delete account Parent
9. Update account Parent
10. View account Parent

## Stack
1. Runtime Environment: `Node.js`
2. Language: `TypeScript`
3. Web Server: `Fastify`
4. ORM: `Prisma`
5. DB: `Postgres`
